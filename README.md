<h2>Projenin Tanıtımı ve Amacı</h2>
<p>Arşiv Yönetim Sistemi adlı proje NYP23-1 grubu olarak Beyza Girgin, Merve Subaşı ve Şeyma Seda Yükseloğlu tarafından oluşturulmuştur.</p>
